Secure your accounts and devices with the right tools. This module focuses on practical defense against active online threats.

1. Introduction: Building Your Defenses

Digital security is the practice of protecting your data, devices, and accounts from unauthorized access. The online world is filled with threats, from simple spam to sophisticated phishing attacks and malware. Our goal is to build your defense systems.

The Three Pillars of Defense:

Effective security relies on a three-pronged approach: **strong authentication** (passwords/2FA), **awareness** (spotting scams), and **secure connections** (Wi-Fi/VPNs).

2. Strong Passwords & Managers

A strong password is long, unique, and complex. Avoid common words, names, or simple keyboard patterns (like "qwerty"). Password strength is measured by the time it takes a computer to guess it.

Using a Password Manager:

The only way to maintain hundreds of unique, complex passwords is by using a dedicated Password Manager (e.g., Bitwarden, 1Password). These tools securely store, generate, and auto-fill login credentials.

  1. **Generate:** Use the manager to create 16+ character passwords.
  2. **Store:** Save the password immediately‚Äîyou never have to memorize it.
  3. **Protect:** Only the master password unlocks your vault.

3. Two-Factor Authentication (2FA)

Two-Factor Authentication adds a second, critical layer of security. Even if a hacker steals your password, they cannot access your account without a second device (the second factor).

Types of 2FA (Best to Worst):

  • **Authentication App (Best):** Uses a time-based code generated by an app like Authy or Google Authenticator. Very secure.
  • **Security Key (Best):** A physical USB device (like YubiKey). Most secure option.
  • **SMS Text Message (Worst):** Codes sent via text are vulnerable to 'SIM swapping' attacks. Only use this if no other option is available.

  • **Suspicious Sender:** Look for misspelled email domains (e.g., `paypalserv.co.uk`).
  • **Generic Greeting:** Real companies usually use your name.

5. Securing Your Home Wi-Fi Network

Your Wi-Fi router is the gateway to your home network. An unsecured router exposes every connected device to risk.

  1. **Change the Default Password:** Routers come with easy-to-guess default passwords (e.g., "admin"). Change both the Wi-Fi network password and the router login password immediately.
  2. **Use WPA3 Encryption:** This is the newest and most secure encryption standard. If WPA3 isn't available, use WPA2-AES. **Never use WEP.**
  3. **Keep Firmware Updated:** Router manufacturers regularly release software (firmware) updates to fix security vulnerabilities. Install them promptly.

6. Summary & Advanced Tools

By using a password manager, enabling 2FA everywhere, and staying alert to phishing attempts, you have established a strong baseline for safe browsing.

For advanced security, consider using a **VPN (Virtual Private Network)**, especially when connecting to public Wi-Fi, to encrypt all your internet traffic.
